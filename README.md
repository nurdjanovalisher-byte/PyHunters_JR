# Telegram AI Bot (aiogram + OpenAI)

Многофункциональный Telegram-бот на Python с интеграцией ChatGPT, викторинами, фактами и интерактивными диалогами.

Проект разработан в учебных целях в рамках обучения на платформе **JavaRush (Python University)**  
с использованием библиотеки **aiogram 3.24.0** и **OpenAI API**.


---

## Возможности бота

- /start — главное меню  
- /random — получение случайного интересного факта  
- /gpt — диалог с ChatGPT  
- /talk — разговор с выбранной знаменитостью  
- /quiz — викторина с подсчётом очков  

---

## Особенности

- FSM (Finite State Machine) для управления состояниями
- Inline-меню и кнопки
- Защита от пустых сообщений
- Индикация "typing..."
- Логирование действий пользователей
- Адаптирован под aiogram 3.x

---

## Используемые технологии

- Python 3.11+
- aiogram 3.24.0
- OpenAI API
- aiohttp
- FSM (Finite State Machine)

---

## Структура проекта
```
ai_open/ 	- работа с OpenAI
handlers/ 	- обработчики команд, inline и FSM
keyboards/ 	- inline и reply клавиатуры
resources/ 	- изображения и текстовые файлы
utils/ 		- вспомогательные модули

main.py
config.py
requirements.txt
.gitignore

```

## Установка и запуск

1. Клонировать репозиторий:
```
git clone <repository_url>
```

2. Установить зависимости:
```
pip install -r requirements.txt
```

3. Создать файл `.env` и указать:
```
BOT_TOKEN=ваш_токен_бота
OPENAI_API_KEY=ваш_openai_api_key
```

4. Запустить бота:
```
python main.py
```
